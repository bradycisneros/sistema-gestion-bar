# Sistema de Gestión para Establecimiento Restaurantero (Bar) 🚀

Este es un proyecto real en desarrollo enfocado en resolver las necesidades operativas de un negocio local, optimizando el control de inventarios, comandas y flujos de caja.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3, JavaScript (JS)
* **Backend:** PHP / Node.js (lo que aplique)
* **Base de datos:** MySQL / PostgreSQL

## 📌 Características principales (En desarrollo)
* Control de stock e insumos en tiempo real.
* Gestión y registro de comandas por mesa.
* Módulo de administración para reportes de ventas diarios.

## ⚙️ Estado del proyecto
Actualmente el proyecto se encuentra en **fase de desarrollo (Beta)**, estructurando la lógica de la base de datos y las conexiones del backend.
